# Job-Seeker
> A basic job search site where users can find jobs
> and apply them.

## Project structure
- User site (Angular)
- Admin site (Angular)
- API (Laravel)

## Included functions
- users can create profile,edit profile, search jobs and apply th
- admin can post jobs , edit jobs, view applied users
